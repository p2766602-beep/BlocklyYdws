# 演算法密室逃脫（YDWS-EscapeRoom）圖像素材清單

> 2026-07-24建立。使用者提議以「霍格華茲魔法學院」意象作為視覺風格方向，本文件評估後列出
> 建議產出的圖像清單，供使用者帶去生圖AI平台產出，再交回來由Claude接進程式碼。

## 重要提醒：風格受啟發、不做官方IP的直接複製

「霍格華茲」「魔法學院」是華納兄弟/J.K.羅琳持有版權與商標的作品，即使是非商業教育用途，
**直接重製官方城堡外觀、學院院徽、原著角色（鄧不利多、麥教授等）、官方配色系統，仍有版權/
商標風險**，尤其本專案是公開部署在GitHub Pages上的網站。

現有的AI prompt設計（`workers/escape-engine/src/prompts.js`裡的`PERSONA_INSTRUCTION`）本來
就只用「魔法學院密室關主」這種**通用奇幻描述**，從來沒有直接寫「霍格華茲」這個詞，這次的圖像
素材建議延續同一個做法：

- 生圖時**不要**在prompt裡打「Hogwarts」「Harry Potter」「Gryffindor」等官方名詞或角色名。
- 改用通用奇幻魔法學院的意象描述（石造城堡、蠟燭吊燈、羊皮紙卷軸、魔杖、水晶球、貓頭鷹、
  古銅色天文儀……），這些是奇幻類型的通用視覺母語，不是霍格華茲專屬的受保護元素。
- 院徽/紋章類素材建議設計成**原創紋章圖案**（例如貓頭鷹＋交叉魔杖、星圖＋書本），不要沿用
  四學院（Gryffindor/Slytherin/Ravenclaw/Hufflepuff）的顏色與圖騰組合。

以下清單的「風格描述」欄位已經照這個原則寫成可以直接丟給生圖AI的通用奇幻魔法學院提示詞，
使用時可以直接使用，不需要另外加上官方作品的名詞。

## 統一風格基準（所有圖像共用，確保彼此搭在一起像同一套）

- **整體調性**：神秘但溫暖鼓勵、莊重不陰森，符合現有AI關主人設「神秘又充滿鼓勵的Game
  Master」的語氣。
- **色調**：暖金色（燭光/羊皮紙）＋深藍紫色（夜色/魔法）為主軸，呼應現有介面配色（背景深藍
  `#0f172a`、強調色紫藍`#6366f1`、琥珀`#fbbf24`、翠綠`#22c55e`、淺紫`#a78bfa`），避免鮮豔
  卡通色，讓生成的圖能跟現有UI融合，不會突兀。
- **畫風**：手繪水彩／復古插畫風，帶一點油畫質感的筆觸，不要用扁平向量卡通風（跟現有emoji
  icon的風格區隔開，圖像素材要有「手作、有溫度」的質感）。
- **格式**：頭像/徽章/邊框類需要**透明背景PNG**（要疊加在深色卡片上）；背景紋理類則是**不透明、
  可無縫重複拼貼（tileable）**。

## 四、進階選配（工作量較大，建議先做完一、二、三再評估要不要投入）

13個主題節點目前用emoji（🔼🔽🔍🫧🎯📥➗🌀🔀🪙🕳️🌊🧩），**可以維持不變**，也可以全部換成
統一風格的客製小圖示。因為數量多（13張）、之後若主題內容或順序調整還要跟著補圖，建議評估
完一～三項的實際效果後再決定要不要做這一批，先在此列出對照表供參考：

| 主題代碼 | 中文名稱 | 建議圖像意象 |
|---|---|---|
| max_value | 找最大值 | 發光的水晶球，越亮代表數值越大 |
| find_min | 找最小值 | 縮小鏡，或一顆最小的星星被圈起來 |
| linear_search | 線性搜尋 | 一列燭火中，用放大鏡逐一檢視 |
| bubble_sort | 氣泡排序 | 一串會自己冒泡上浮排列的魔法泡泡 |
| selection_sort | 選擇排序 | 一頂會自動挑出最適合的分類帽 |
| insertion_sort | 插入排序 | 一本書架，書一本一本被插入正確位置 |
| binary_search | 二分搜尋 | 一個會自動對半打開指向正確那一半的魔法書 |
| recursion_basics | 遞迴基礎 | 俄羅斯娃娃式的魔法箱子，一層打開又一層 |
| merge_sort | 合併排序 | 兩瓶不同顏色的藥水合併成一瓶漸層藥水 |
| greedy | 貪婪演算法 | 一堆金幣，手總是先抓最大的那一枚 |
| dfs | DFS深度優先 | 一支蠟燭沿著迷宮牆壁一路探到底 |
| bfs | BFS廣度優先 | 水面上一圈一圈擴散開的魔法水波紋 |
| dp | 動態規劃 | 一本會自動記錄先前答案的魔法筆記本 |

如果之後要做這批，每張規格建議跟第二節的Tier icon一致：48×48px、透明背景、跟其他素材同一套
暖金/深紫水彩插畫風。

## 技術提醒

- `YDWS-EscapeRoom`是獨立repo、獨立部署，圖檔請放進這個專案自己的`public/`資料夾，跟
  BlocklyYdws的`public/avatars/`（AI伴學頭像）是不同的實體檔案，即使風格類似也要各自準備。
- 頭像/徽章/邊框類記得輸出**透明背景**（PNG），背景紋理類記得做成**可重複拼貼**，這兩點如果
  生圖平台一開始沒做到，之後可能需要另外用去背工具處理，先跟生圖平台確認有沒有透明背景選項。
- 檔案大小盡量壓縮（比照現有AI伴學頭像從1.5MB壓到約52KB的做法），避免拖慢網站載入速度，
  交回來的時候可以先問我要不要幫忙再壓縮一次。
